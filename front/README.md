# 🎯 Front

## 🛠 Installation

#### 📦 With Docker (easy way)

```console
$ docker-compose build; docker-compose up -d
```

#### 🤯 Without Docker (hard way)

###### 🔎 Requirements

- NodeJS installed

####### ✅ Installation

```console
$ npm i
```

####### 🚀 Run

Basic run:

```console
$ npm start
```
